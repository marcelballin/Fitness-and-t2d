# Adolescent cardiorespiratory fitness and future type 2 diabetes
#### Last modified: 2025-03-31
#### Author: Marcel Ballin

This repository contains the analytical code files associated with Ballin et al. (2025), investigating adolescent cardiorespiratory fitness and adult type 2 diabetes using Swedish military conscription data. For more details, refer to the preprint available on medRxiv: https://www.medrxiv.org/content/10.1101/2024.11.26.24318038v1
